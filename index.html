01 · Projet principal

GE
GestEmprunt

GestEmprunt – Gestion de prêts de matériel

↳ Réalisé dans le cadre du stage — Lycée Henri Bergson

Application web complète de gestion d'emprunts de matériel pour un établissement scolaire. Développée en PHP/MySQL avec architecture rôle-based (admin, professeur, observateur), notifications SMTP, calendrier interactif et système de réservations avec validation admin.

  • Inscription professeur avec validation administrateur
  • Rôle observateur pour comptes en attente
  • Calendrier couleur des emprunts et réservations (calendrier.php)
  • Seuils d'alerte de stock avec surlignage rouge et tri tableau de bord
  • Notifications par e-mail via PHPMailer (SMTP)
  • Conversion automatique réservation → emprunt validé
  • Bouton « marquer comme rendu » pour l'admin
  • Bilan récapitulatif des emprunts
PHP MySQL / PDO PHPMailer HTML/CSS JavaScript RBAC SMTP

02 · Projet scolaire

MissionStage – Gestion internat & sorties

↳ Réalisé dans le cadre du stage — Lycée Henri Bergson

Application PHP de gestion des autorisations de sortie pour élèves internes. Insertion par plage de dates, profils élèves, sauvegarde BDD et authentification avec rôles (loge, CPE, administrateur).

  • Saisie / modification / annulation d'autorisations
  • Historique par élève, import CSV
  • Hachage bcrypt des mots de passe
  • Interface refaite : topbar + sidebar + fil d'Ariane
PHP MySQLi Latin-1 Sessions bcrypt

04 · Exploration technique

NetLock RMM – Déploiement open source

Compilation et déploiement d'une solution RMM open source (.NET/Blazor + MySQL) dans un environnement lycée pour gestion de ~1000 postes. Résolution d'erreurs d'auth MySQL, génération de stubs, rédaction d'un guide de déploiement en français.

.NET / Blazor MySQL SQLite RMM Windows
